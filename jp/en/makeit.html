For example, we will make something like this

Cloud

By using the cloud, we create a flexible system that is not restricted to places such as offices, stores, and places to go, as well as PC and smartphones.

Handy terminal

By using a handy terminal to directly manage the materials and products in front of you, we will create a system that is easy to see visually and that prevents work leakage.

IC tag

By using IC tags and readers, we will create a system that manages the quantity and location of goods, as well as security and records of office entry and exit.

Camera

We will create a system that monitors new borns, elderly people, pets, etc. using cameras, and detects people, vehicles, etc. by image analysis.

Factory / Warehouse

We create systems for factories and warehouses, such as transport control of automatic warehouses, management of incoming/outgoing/inspection, and creation of slips using label printers.

Solar Panel

We will create a management system that automatically collects information from solar power station equipment, and then make the graphs, displays it, and analyzes it.

Welfare services

We will create a system, which will not only manages welfare services and users, but also create reports to be submitted to the government according to the services, and automatically calculate various insurances and compensations.

Smartphone app

In the form of smartphone applications, we will create systems that can be used not only for business systems, but also for GPS-based services, family watching, and fieldwork, etc.

Robot

We will create control systems such as interactive customer service by robots and advertisements that will draw the attention with unique movements such as dancing.
